Hi, I am Pierre-Alexandre Mousset - Software Developer based in Boston

I am currently pursuing a master's degree in Computer Science at Northeastern university. Prior to Northeastern, I spent two years as a reservist for the French army and worked in the finance industry. I am looking to build challenging and impactful products.


My first programming experience was at Asia-HR where I designed, developed and deployed a job-search platform. After working on ClimateTree with my classmates, I am now helping my parents transition from a brick-and-mortar to an online business.

Plagius - Python & JS Plagiarism Checker

Plagius is a code plagiarism checker for code written in Python 3 and JavaScript (ES6). It is built using Maven Framework and uses ANTLR parser generator for parsing. The user submit two source files before to obtain an estimated percentage of plagiarism between those two files. Website coming soon
